# Fiscal Patriot Dashboard - Tableau Workbooks

Tableau visualizations for the AGA Datathon 2026, analyzing federal audit and spending data.

---

## FAC_Data_Visuals_Workbook
Federal Audit Clearinghouse single audit data (FY2016-2024).

### Visualizations
- Material Weakness Findings by State (map + bar)
- Total Findings by State (map + bar)
- Significant Deficiency Findings by State (map + bar)
- Material Weakness Rate by State (map)

---

## FAC_USAspending_Merge_FY2023-2024_Workbook
Merged Federal Audit Clearinghouse and USASpending data (FY2023-2024).

### Visualizations
- Findings Per $1M in Federal Spending by State (map)
- Top 20 Recipients by Audit Findings (bar)
- Findings by Entity Type (bar)
